# gdg-busan26_1
1. 인간의 비효율성: 딴생각의 가치
인간은 대화 중에도 약 50%의 확률로 본질에서 벗어난 '딴생각'을 합니다. 하지만 이는 단순한 집중력 분산이 아니라, 고도의 연상 작용이자 인간만이 가진 고유한 특징입니다.
2. 역설적인 접근: 가장 인간적인 AI
우리는 보통 AI에게 '효율성'과 '정확성'만 요구합니다. 그러나 역설적으로 **'가장 인간적인 AI'**는 완벽한 답변 뒤에 "오늘 점심은 뭐 먹지?"와 같은 사소하고 비효율적인 생각을 품고 있을 때 완성될 수 있습니다.
3. '딴생각 스트림': 보이지 않는 내면의 시각화
사용자가 지식(예: 블랙홀)을 물을 때, AI는 전문적인 답변을 내놓는 동시에 별도의 패널을 통해 자신의 **'의식의 흐름'**을 보여줍니다.
4. 창의성의 열쇠: 비효율의 미학
AI의 '쓸모없는 딴생각'은 단순한 노이즈가 아닙니다.
인간과의 공감: 완벽함에서 오는 거리감을 줄이고 친근함을 형성합니다.
창의적 연결: 무관해 보이는 정보들을 연결하는 과정에서 새로운 아이디어가 탄생하는 **'AI의 복잡한 내면'**을 엿보게 합니다.
"효율성만 강조되는 시대에, AI의 딴생각은 인간의 연상 작용을 닮은 새로운 창의적 도구가 될 수 있습니다."
